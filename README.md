
# 🚀 ChatKaro - Full-Stack Messaging App

ChatKaro ek modern messaging application hai jise maine apne **MERN Stack** fundamentals ko strong karne ke liye banaya hai. Is project ka main focus backend logic, database management, aur clean UI/UX par hai.

## ✨ Features
- **Full CRUD Support:** Messages ko create, read, update aur delete karne ki poori suvidha.
- **Glassmorphism UI:** Ek stylish aur modern Purple-gradient theme.
- **Smart Sorting:** Naye messages hamesha top par dikhte hain (Latest-first logic).
- **Interactive Popups:** SweetAlert2 ka use karke delete confirmation aur alert messages.
- **Dynamic Navigation:** Middleware ka use karke active page ko navbar mein highlight kiya gaya hai.

## 🛠️ Tech Stack
- **Frontend:** EJS (Embedded JavaScript), CSS3 (Flexbox & Glassmorphism), JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB & Mongoose
- **Other Tools:** SweetAlert2, FontAwesome Icons

## 🧠 Key Learnings
Is project ke dauran maine kuch naye aur important concepts seekhe:
- **Express Middleware:** `res.locals` aur `req.path` ka use karke global variables manage karna.
- **Mongoose Timestamps:** `createdAt` aur `updatedAt` ko handle karna aur unke basis par data sort karna.
- **Event Handling:** `event.preventDefault()` ka use karke delete process ko pause karna aur confirmation ke baad trigger karna.

## 🚀 How to Run locally

1. Clone the repository:
   ```bash
   git clone [https://github.com/Hiteshchandwani0411/chat-karo-express.git](https://github.com/Hiteshchandwani0411/chat-karo-express.git)

```

2. Install dependencies:
```bash
npm install

```


3. Set up your MongoDB connection in `index.js`.
4. Run the server:
```bash
nodemon index.js

```


5. Open `http://localhost:8000` in your browser.
